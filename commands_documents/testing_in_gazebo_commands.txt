tap 1 — world file neerzetten

cp ~/Downloads/cogbot_world ~/ros2_ws/src/mirte-gazebo/worlds/cogbot_world.world
Stap 2 — Gazebo starten met jouw world

cd ~/ros2_ws
source install/setup.bash
ros2 launch mirte_gazebo gazebo_mirte_master_empty.launch.xml \
  world:=$(pwd)/src/mirte-gazebo/worlds/cogbot_world.world
Stap 3 — Je service starten in een tweede terminal

source ~/ros2_ws/install/setup.bash
ros2 run cognitive_robot detect_abacus_service \
  --ros-args -p camera_topic:=/camera/image_raw
Let op de camera topic: in Gazebo is dat /camera/image_raw, op de echte robot /camera/color/image_raw.

Stap 4 — Service handmatig aanroepen om te testen

source ~/ros2_ws/install/setup.bash
ros2 service call /detect_abacus cognitive_robot_interfaces/srv/DetectAbacus {}
Controleren welke camera topics beschikbaar zijn
Als je twijfelt over de juiste topic naam in jouw world:


ros2 topic list | grep camera
kan ik dan ook de robot zelf bewegen?
Ja, je kunt de robot besturen via het toetsenbord in een aparte terminal:


source ~/ros2_ws/install/setup.bash
ros2 run teleop_twist_keyboard teleop_twist_keyboard \
  --ros-args --remap cmd_vel:=/mirte/cmd_vel
