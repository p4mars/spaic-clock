Unit tests for the cognitive_robot package, run by `colcon test`.

* test_detect_abacus_service.py — real unit tests for the abacus detection
  service (fully mocked: no robot, camera, or internet needed).
* test_copyright.py / test_flake8.py / test_pep257.py / conftest.py —
  standard ROS 2 boilerplate (license + style linters), auto-generated by
  `ros2 pkg create`.

Run them with:
    colcon test --packages-select cognitive_robot
or directly:
    python3 -m pytest cognitive_robot/test -v
